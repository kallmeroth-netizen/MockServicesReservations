# Event photography

These three photos are referenced by `index.html` (via the `.ed-bg` and
`.confirm-photo` styles). Drop the final image files in this folder using the
exact filenames below and they will appear automatically — no code changes
needed. Until the files are added, each panel falls back to its dark gradient.

| File                          | Where it shows                | Suggested shot              | Orientation        |
|-------------------------------|-------------------------------|-----------------------------|--------------------|
| `01-storefront.jpg`           | Welcome / landing hero        | Storefront w/ orange awning | Portrait (tall)    |
| `02-pizza.jpg`                | Booking flow hero             | Pizza box close-up          | Portrait (tall)    |
| `03-storefront-closeup.jpg`   | Confirmation page photo       | Storefront close-up + sign  | Landscape (wide)   |

Tips:
- Keep them reasonably optimized (~200–500 KB each) so the page stays fast.
- The panels use `background-size: cover`, so any aspect ratio works; the
  orientation notes above just give the best framing for each slot.
